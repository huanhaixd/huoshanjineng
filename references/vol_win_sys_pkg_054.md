
## 火山"视窗项目"系统类库模块: WinHTTP5_1

### 包: 火山.WinHTTP5_1
* **描述:** WinHTTP 5.1仅适用于Windows Server 2003, Windows 2000 Professional with Service Pack 3 (SP3), Windows XP with Service Pack 1(SP1)及更高版本的操作系统.

---

#### 火山.WinHTTP5_1.WinHttp选项

* **类型:** `类`
* **名称:** `WinHttp选项`
* **特性:**
	* `常量类`(对应数据类型: `整数`)
* **文档分类:** `辅助类`
* **描述:** 设置选项

##### 火山.WinHTTP5_1.WinHttp选项.用户代理

* **类型:** `成员常量`
* **名称:** `用户代理`
* **数据类型:** `WinHttp选项`
* **描述:** 用户代理

##### 火山.WinHTTP5_1.WinHttp选项.重定向网址

* **类型:** `成员常量`
* **名称:** `重定向网址`
* **数据类型:** `WinHttp选项`
* **描述:** 检索包含资源的链接.此值为只读,在调用"打开"方法之前,无法使用.此选项可用于检查"发送"方法完成后的网址,以验证是否发生了任何重定向.

##### 火山.WinHTTP5_1.WinHttp选项.编码

* **类型:** `成员常量`
* **名称:** `编码`
* **数据类型:** `WinHttp选项`
* **描述:** 设置或检索标识字符串的编码方式.默认值为UTF-8编码方式.代码页用于将"打开"方法中所传递的字符串转换为单字节字符串表示形式.

##### 火山.WinHTTP5_1.WinHttp选项.网址转义符

* **类型:** `成员常量`
* **名称:** `网址转义符`
* **数据类型:** `WinHttp选项`
* **描述:** 指示网址字符串中的百分比字符是否转换为转义序列.默认值为真,它指定所有ANSI字符,但百分比符号转换为转义序列除外.

##### 火山.WinHTTP5_1.WinHttp选项.忽略证书

* **类型:** `成员常量`
* **名称:** `忽略证书`
* **数据类型:** `WinHttp选项`
* **描述:** 指示应忽略哪个服务器证书错误

##### 火山.WinHTTP5_1.WinHttp选项.SSL证书

* **类型:** `成员常量`
* **名称:** `SSL证书`
* **数据类型:** `WinHttp选项`

##### 火山.WinHTTP5_1.WinHttp选项.证书检查

* **类型:** `成员常量`
* **名称:** `证书检查`
* **数据类型:** `WinHttp选项`
* **描述:** 在SSL协商期间启用服务器证书吊销检查.当服务器提供证书时,将执行检查以确定证书是否已由其颁发者吊销.如果证书确实被吊销,或者吊销检查失败,因为无法下载证书吊销列表,请求将失败.无法取消此类吊销错误.
